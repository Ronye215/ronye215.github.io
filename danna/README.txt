
# Invitación | Cumpleaños de Danna

Cómo usar:
1) Abre index.html en tu navegador o súbelo a tu hosting.
2) Personaliza el enlace con parámetros:
   - ?name=Tu%20Nombre
   - &msg=Un%20saludo%20opcional
   - &rsvp=NUMERO_WHATSAPP (solo dígitos, ej. 50255500000)

Ejemplo: ./index.html?name=Karla&msg=¡Te%20esperamos!&rsvp=50255500000

Editar datos del evento:
- Dentro de index.html, busca el objeto EVENT en el <script> y cambia fecha, hora, etc.

Archivos:
- index.html: Página completa del prototipo.
